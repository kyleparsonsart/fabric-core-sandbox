🎨

Role-based semantics

Tokens describe purpose (fg/neutral) not appearance. Theming changes semantics, not the token names.

🌓

Dark mode–ready

All semantic tokens have light/dark assignments. Switch themes by remapping primitives, no rewrites.

🔗

Primitive traceability

Every token traces back to a v3 stone or sun primitive — no orphaned hex values.

⚠️

Raw value detection

The panel flags elements using hardcoded hex or rgba values instead of tokens — no token name found.
